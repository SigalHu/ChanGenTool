# ChanGenTool
信道建模与仿真软件
## 准备
* VS2010
* MATLAB R2013a x64
* [PCIEDriver](https://github.com/SigalHu/PCIEDriver)
## 实现功能
根据用户输入参数生成动态或静态信道，并通过PCIE将信道数据发送到底层设备
